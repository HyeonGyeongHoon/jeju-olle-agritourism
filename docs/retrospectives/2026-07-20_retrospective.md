## 4. 추가 진행: 지도 이미지 기반 코스 메타데이터 정합성 검수 및 교정
`data/map_images/` 내 코스별 인포그래픽 지도 원본을 페이지 단위로 직접 대조하여, PDF 텍스트 추출 과정에서 발생한 `courses_metadata.json`의 순서 오류 및 깨진 지점명을 **3코스부터 21코스까지 전 구간** 교정함 (1, 1-1, 2코스는 이전에 선(先) 교정 완료된 상태).

### 4.1. 교정 대상 및 방식
- 각 코스의 지도 이미지(예: `3코스_p040.png`, `10-1코스_p076.png` 등)를 직접 열람하여 시작점(S)-종점(F) 간 실제 경유지 순서와 누적 거리(km)를 눈으로 재확인하고, 기존 자동 파싱값과 1:1 대조함.
- PDF 파서가 페이지 푸터/범례("코스의 높낮이", "etuoR ellO ujeJ" 역순 워터마크 등)를 지점명으로 잘못 추출하거나, 두 지점 이름이 한 줄에 걸쳐 잘림·중복되는 경우를 다수 발견하여 실제 지도 라벨 기준으로 전량 교체함.

### 4.2. 코스별 주요 교정 내역
- **3-A/3-B코스**: 한 장의 지도에 A·B 두 경로가 겹쳐 표기된 구조를 확인하여, 온평포구→A·B갈림길(0.5km) 분기 및 신풍포구(A·B코스 이음길)에서의 재합류 지점을 반영해 두 코스의 구간 거리를 독립적으로 재구성.
- **15-A/15-B코스**: 마찬가지로 한림항→A·B갈림길(1.6km) 분기 후 고내포구에서 합류하는 구조를 확인, A(16.5km)/B(13.0km) 각각의 실제 경유 지점으로 전면 교체.
- **7, 8, 10-1, 13, 14, 14-1, 16, 17, 18-2코스**: `start_point`/`end_point`가 지도상 실제 S/F 라벨과 불일치하던 오류(예: 18-2코스는 시작·종점이 서로 뒤바뀌어 있었음) 수정.
- **13코스**: `total_distance_km`이 지도 기준 총 길이(16.2km)와 달라(기존 15.9km) 있던 것을 수정.
- **각 코스 공통**: `lunch_info` 등 텍스트 필드 말미에 남아있던 페이지 워터마크 잔재(예: "6 코스 완주 스탬프 : 섶 섬 60 etuoR ellO ujeJ")를 제거하고, `stamp_locations`는 지도상 스탬프 아이콘 위치와 재대조하여 검증(6코스의 "소정방폭포 소라의 성" 등 깨진 표기는 "소천지"로 정정).

### 4.3. 검증
- 교정 완료 후 `python -c "import json; json.load(...)"` 로 JSON 구조 유효성을 확인, 총 29개 코스 항목(1~21코스 및 하위 A/B/지선 코스 포함) 정상 파싱 확인.
